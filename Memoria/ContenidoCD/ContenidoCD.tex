\chapter{Contenido del CD adjunto}

Con la presente memoria se adjunta un CD-ROM con los siguientes contenidos:

\begin{enumerate}
	\item Esta documentación en formato PDF para su cómoda visualización, así como los ficheros utilizados para su creación con \LaTeX, todo ello dentro de la carpeta \emph{Memoria}.
	\item Todos los componentes software necesarios para instalar (y compilar en caso de que se quisiese) la aplicación: Java JDK, JBoss, PostgreSQL, JDOM, Jakarta Struts, JSTL, todo ello dentro de la carpeta \emph{Programas}.
	\item Los ficheros necesarios para ejecutar la aplicación sin instalar: el fichero .ear de la aplicación, los scripts sql para la creación y borrado de tablas en la base de datos (PostgreSQLCreateTables.sql y PostgreSQLDeleteTables.sql), y el fichero postgres-ds.xml que se debe copiar en el directorio de JBoss \emph{server/default/deploy/} al igual que el fichero .ear.
	\item El código fuente de la aplicación dentro de la carpeta \emph{MapViewer-src-1.0} y los diagramas UML en formato MagicDraw, dentro de la subcarpeta \emph{Diagramas}.
\end{enumerate}
