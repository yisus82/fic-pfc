\section{J2EE Examples}
Los ficheros \emph{StandarUtil.jar} y \emph{WebUtil.jar} empleados en este proyecto forman parte de los ejemplos \emph{J2EE-Examples} del profesor Fernando Bellas Permuy. Estos ejemplos se emplean en la asignatura de Integración de Sistemas que se imparte en la Facultad de Informática de la Universidade da Coruña. 

\section{Javascript Image Cropper}
Estas librerías Javascript se usan para hacer la selección de una zona geográfica en un mapa.

\section{WMSMap}
Estas librerías Javascript se usan para facilitar el manejo de los mapas y las diversas peticiones a los WMSs.

\section{TinyMCE}
Este editor HTML en Javascript se usa para facilitar la introducción del contenido HTML de los POI (Points Of Interest, Puntos de interés).
